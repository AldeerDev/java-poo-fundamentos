# Leccion 06 - Interfaces

## Objetivo

Comprender cómo utilizar interfaces para definir comportamiento en Java.

---

## Conceptos clave

- Interface
- implements
- Contrato
- Polimorfismo
- Múltiple implementación

---

## ¿Qué es una Interface?

Es un contrato que obliga a las clases a implementar ciertos métodos.
